# Redbud Kitchen — Hill Country Restaurant Landing Page

> Demo project built by [Mulberry Cloud LLC](https://mulberrycloud.com) to showcase small business web development capabilities. Built with vanilla HTML/CSS/JS, deployable as a static site.

---

## About This Demo

A polished, fully responsive landing page for **Redbud Kitchen** — a fictional farm-to-table restaurant in Wimberley, TX. This project demonstrates what Mulberry Cloud can build for local restaurants and hospitality businesses.

## Features

- **Hero section** with full-width imagery, tagline, and CTA
- **About section** with warm, personal brand storytelling
- **Seasonal menu** with tabbed Starters / Mains / Desserts
- **Hours & Location** with address, hours table, and map placeholder
- **Reservation form** with client-side validation and confirmation toast
- Fully **mobile responsive** with hamburger nav
- Scroll-reveal animations via IntersectionObserver

## Design

- Warm earthy palette — cream, terracotta, deep olive
- Serif headings via **Playfair Display**, clean body via **Lato**
- No external image dependencies — all placeholders use CSS gradients

## Deploy

This repo includes a `render.yaml` for one-click deployment to [Render](https://render.com):

1. Fork or clone this repo to your GitHub account
2. Connect repo to Render → **New → Static Site**
3. Render auto-detects `render.yaml` — hit **Deploy**

PR preview URLs are enabled by default.
